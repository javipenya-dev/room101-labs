## 📌 Overview

This repository is my **personal sandbox**: a place for experiments, ideas, tests, notes and anything I want to explore freely.

It is intentionally chaotic, creative and open‑ended.  
A space to try things, break things, learn things — without pressure.

---

## 📂 Repository Structure

```
room101-labs/
│
├── experimentos/
├── ideas/
├── misc/
├── notas/
├── scripts/
├── tests/
└── trash/
```

### Folder descriptions

- **experimentos/** → Experiments, prototypes and technical curiosities  
- **ideas/** → Future concepts, project seeds and inspiration  
- **misc/** → Random things I want to keep  
- **notas/** → Quick notes, thoughts and reminders  
- **scripts/** → Small scripts and utilities  
- **tests/** → Tiny tests, proofs of concept and throwaway code  
- **trash/** → Things that shouldn’t exist… but do  

---

## 🎯 Purpose

To keep a creative space where I can explore freely, learn by doing and store anything that doesn’t belong in my professional repos.
